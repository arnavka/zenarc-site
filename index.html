Zenarc Minecraft Server

This subdomain is used for connecting to a private Minecraft server hosted locally. The server is accessible only to a small group of friends (2–3 people).

We use Cloudflare Tunnel because port forwarding is not possible on our network. The tunnel securely routes traffic to our game server without exposing our home IP address.

Server Address (private use only): zenarc.is-a.dev

⚠️ This server is not open to the public. It’s a hobby project for personal learning and private multiplayer gaming.
